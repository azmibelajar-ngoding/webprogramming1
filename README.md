# webprogramming1
Tugas 1 Pemrograman Berbasis Web | Sistem Informasi - Universitas Terbuka

Ini merupakan proyek Tugas Praktek mata kuliah Pemrograman Berbasis Web, di sini saya mempraktekkan pembuatan web menggunakan HTML, CSS dan JavaScript DOM.

Gambaran Website:
1. Website profil/portofolio
2. Berisi foto profil, title, navbar, dan form.

Fitur-fitur:
1. Tersedia Hamburger Menu untuk mobile
2. Autoscroll (Smooth Scroll) ketika menekan menu navigasi.
3. Back to Top Button untuk memudahkan kembali ke atas.
4. Popup Ringkasan Profil menggunakan Modal.
5. Popup validasi formulir.
6. Scroll Progress Bar di bagian atas halaman.

Masih proses:
1. Penyempurnaan desain UI untuk popup validasi di CSS.
2. Melengkapi konten teks pada beberapa section yang masih kosong.
3. Penambahan file php untuk menyimpan data formulir.

Update:
1. Perbaikan bug formulir di Hire Me.

Cara menjalankan:
1. Clone repositori ini (git clone https://github.com/azmibelajar-ngoding/webprogramming1)
2. Download file zip
3. Extrak ke penyimpanan perangkat
4. Pastikan semua file berada di dalam 1 folder yang sama
5. Buka file profile.html menggunakan browser

Folder Structure:
+ /profile.html —————————— main file
+ /styles.css    —————————— styling
+ /script.js    —————————— logic
+ /Image/profilepic.jpeg —————————— photo asset
